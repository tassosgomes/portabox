# UI Kit — Admin Dashboard

Web dashboard for **síndicos** (condo managers). KPIs, package throughput over
time, and a searchable list of deliveries. 1280px design width. Navy sidebar +
paper canvas.

> Note: source materials include only a promotional render of this surface. The
> layout below is a first-pass interpretation and should be validated with real
> users.

## Components
- `AdminShell` — sidebar + top bar
- `KPICard` — single metric with trend
- `DeliveriesChart` — simple bar chart (SVG, no dependency)
- `PackageTable` — searchable, filterable list
