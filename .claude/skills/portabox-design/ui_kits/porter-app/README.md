# UI Kit — Porter App (PortaBox Capture)

Mobile-first app used by **porteiros** at the front desk to photograph package
labels. The AI/OCR backend extracts apartment, resident name, and carrier
from each photo; the porter confirms and the resident is notified.

## Screens
1. **Home** — today's packages, waiting count, big capture CTA
2. **Capture** — camera frame, dashed orange guide, OCR confirmation
3. **Confirm** — extracted fields + resident preview, send notification
4. **Retrieve** — porter enters PIN to release a package to the resident
5. **History** — today's completed handoffs

All pt-BR. iPhone frame. Light paper background with navy top chrome.
